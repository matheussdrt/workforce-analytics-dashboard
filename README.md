# 📊 Workforce Analytics Dashboard (Power BI)

## 📌 Sobre o projeto
Este projeto consiste em um dashboard desenvolvido no Power BI com foco em análise de funcionários (People Analytics), integrando dados de RH e financeiros para geração de insights estratégicos.

## 🎯 Objetivo
Fornecer uma visão clara e interativa sobre a força de trabalho da empresa, auxiliando na tomada de decisões com base em dados.

## 📈 Principais métricas
- Quantidade total de funcionários
- Distribuição por setores
- Custos com pessoal
- Indicadores de desempenho
- KPIs gerais de RH

## 🛠️ Tecnologias utilizadas
- Power BI
- Excel (ou fonte de dados)
- Modelagem de dados
- DAX (Data Analysis Expressions)

## 🔍 Funcionalidades
- Dashboard interativo
- Filtros dinâmicos
- Visualização clara de indicadores
- Análise por setor e desempenho

## 📁 Estrutura
- `dashboard.pbix` → Arquivo principal do Power BI
- `dados/` → Base de dados utilizada (se aplicável)

## 🚀 Como usar
1. Baixe o arquivo `.pbix`
2. Abra no Power BI Desktop
3. Explore os dashboards e utilize os filtros interativos

## 📌 Insights gerados
O dashboard permite identificar padrões de distribuição de funcionários, análise de custos e oportunidades de otimização na gestão de pessoas.
